# Iron Palace

A modern, responsive mock website for a fictional premium gym built with **Next.js**, **React**, and **Tailwind CSS**.

🌐 **Live Demo:** https://iron-palace-henna.vercel.app/

> **Disclaimer:** This is a demonstration project only. Iron Palace is a fictional gym created to showcase web development skills.

## Features

- Responsive design for desktop, tablet, and mobile
- Modern landing page with custom branding
- Membership plans with interactive selection
- Coach directory with dynamic profile pages
- FAQ section
- Event calendar
- Contact and tour request pages
- SEO metadata for each page
- Accessible navigation and semantic HTML
- Optimized images using Next.js Image

---

## Tech Stack

- **Next.js 16**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Vercel** (deployment)

---

## Running Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/iron-palace.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
